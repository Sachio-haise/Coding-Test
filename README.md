## Usage
file copy
```bash
 cp .env.example .env
```

install library
```bash
composer install   
```

generate key
```bash
php artisan key:generate
```

node modules install
```bash
npm install
```

start run node
```bash
npm run dev 
```

start run php
```bash
php artisan serve
```

if you want to use docker install [Sail](https://laravel.com/docs/10.x/sail)
```bash
./vendor/bin/sail up -d
```
